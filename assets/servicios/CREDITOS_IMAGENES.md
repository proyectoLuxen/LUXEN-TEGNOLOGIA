# Creditos de imagenes - servicios tecnicos

Fecha de actualizacion: 2026-06-22

En la V2 no se utilizaron fotografias externas para la seccion de servicios tecnicos.

La pagina `servicios-tecnicos.html` usa iconos tipograficos propios dentro de tarjetas visuales. No se descargaron imagenes de Pexels, Unsplash u otros bancos, y no hay hotlinks a recursos externos.
